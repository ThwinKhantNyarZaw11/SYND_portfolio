BIM Implementation

Project Overview

Led Building Information Modeling (BIM) integration for a mixed-use commercial development, coordinating design and construction documentation across all disciplines. This initiative enhanced project accuracy, reduced clashes, and improved collaboration.

  • Set BIM standards and protocols for file naming, worksets, and clash resolution
  • Managed 4D sequencing with construction simulation using Navisworks
  • Trained stakeholders in model navigation, annotation, and schedule coordination
  • Integrated COBie data for facility management handover
